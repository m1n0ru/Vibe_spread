# 請求書ジェネレータ 要件定義書

## 概要

請求書を画面上で入力・編集し、ExcelファイルとPDFの両形式で出力できるWebアプリケーション。  
技術スタック：Vanilla HTML / CSS / JavaScript（依存ライブラリ最小限）

---

## 機能要件

### 1. 請求書入力フォーム

#### 発行者情報（自社情報）
- 会社名
- 住所
- 電話番号
- メールアドレス
- 振込先銀行情報（銀行名・支店名・口座種別・口座番号・口座名義）

#### 宛先情報（請求先）
- 会社名 / 氏名
- 住所

#### 請求書メタ情報
- 請求書番号（自動採番 + 手動編集可）
- 発行日（日付ピッカー、デフォルト：今日）
- 支払期限（日付ピッカー）

#### 明細行（複数行対応）
| フィールド | 説明 |
|---|---|
| 品目名 | 商品・サービス名 |
| 数量 | 数値入力 |
| 単位 | 個・時間・式など |
| 単価 | 数値入力（円） |
| 金額 | 数量 × 単価（自動計算） |

- 行の追加・削除ボタン
- 明細は最大50行まで

#### 金額集計（自動計算）
- 小計
- 消費税率選択（0% / 8% / 10%）
- 消費税額
- 合計金額

#### 備考欄
- フリーテキスト（複数行）

---

### 2. リアルタイムプレビュー

- 入力内容が即座に請求書プレビューへ反映
- A4縦サイズのレイアウトを画面右側に表示
- 日本語フォント対応

---

### 3. Excel出力（スプレッドシート）

- ボタン1クリックで `.xlsx` ファイルをダウンロード
- 出力内容：
  - シート1「請求書」：請求書と同じレイアウト
  - シート2「明細データ」：明細行をテーブル形式（集計・ピボット向け）
- ファイル名：`請求書_[請求書番号]_[発行日].xlsx`
- ライブラリ：[SheetJS (xlsx)](https://sheetjs.com/) をCDNから読み込み

---

### 4. PDF出力

- ボタン1クリックでブラウザのPDF印刷ダイアログを呼び出す
- または `html2canvas` + `jsPDF` でPDFファイルとして直接ダウンロード
- A4縦レイアウトで出力
- ファイル名：`請求書_[請求書番号]_[発行日].pdf`

---

### 5. データ保存・復元

- 入力内容をブラウザの `localStorage` に自動保存（入力のたびに保存）
- ページ再読み込み時に前回の入力内容を復元
- 「クリア」ボタンで全フィールドをリセット

---

## 非機能要件

| 項目 | 内容 |
|---|---|
| 動作環境 | モダンブラウザ（Chrome / Safari / Firefox 最新版） |
| サーバー不要 | 静的ファイル（HTML + CSS + JS）のみで動作 |
| レスポンシブ | デスクトップ優先（幅1280px基準） |
| 外部依存 | CDN経由でSheetJS・jsPDFのみ |
| 日本語対応 | UI・出力ともに日本語 |

## 画面構成

```
+---------------------------+---------------------------+
|       入力フォーム         |      請求書プレビュー      |
|                           |                           |
|  [発行者情報]              |  ┌─────────────────────┐  |
|  [宛先情報]                |  │      請 求 書        │  |
|  [請求書番号・日付]         |  │  発行日: 2026/06/09  │  |
|  [明細テーブル]            |  │  請求先: ○○株式会社  │  |
|    + 行追加ボタン          |  │  明細...             │  |
|  [消費税・合計]            |  │  合計: ¥100,000      │  |
|  [備考]                   |  └─────────────────────┘  |
|                           |                           |
|  [Excel出力] [PDF出力]     |                           |
|  [クリア]                 |                           |
+---------------------------+---------------------------+
```
